# IT-Lernmanager

Diesen Code habe ich als Übungsaufgabe geschrieben, um meine bisherigen Kenntnisse miteinander zu kombinieren.

Der Benutzer gibt zuerst ein, wie viele Aufgaben ausgewertet werden sollen. Danach wird für jede Aufgabe die erreichte Punktzahl abgefragt.

Das Programm berechnet anschließend:

- die Gesamtpunktzahl
- den Durchschnitt
- die Anzahl der bestandenen Aufgaben
- die Anzahl der nicht bestandenen Aufgaben
- ein Gesamtergebnis

Bei der Bewertung der Punktzahlen habe ich mich an einem einfachen Schulnotensystem orientiert: „Sehr gut“, „Gut“, „Bestanden“ oder „Nicht bestanden“.

## Diese Funktionen und Befehle habe ich benutzt

- Eingaben und Ausgaben mit `input()` und `print()`
- Bedingungen mit `if`, `elif` und `else`
- eine `for`-Schleife mit `range()`
- Variablen und einfache Berechnungen
- Zähler für bestandene und nicht bestandene Aufgaben
